```

[핵심 가치]를 제공한다.
```

### 컨셉

```
이 서비스는 한마디로 ___다.
```

### 포지셔닝 (2단계에서 확정)

```
우리 서비스는 [타겟]을 위한 [카테고리]로,
[경쟁사 또는 기존 방법]과 달리
[핵심 차별점]을 제공한다.
```
