## [P-01] 경쟁사 식별

> **목적:** 직접·간접 경쟁사를 목록으로 만든다.

```
1단계에서 확정한 내용이야:
- 문제 정의: [검증 결과 문서에서 복사]
- 핵심 타겟: [검증 결과 문서에서 복사]
- 서비스 개요: [한 줄 요약]

이 정보를 바탕으로 경쟁사를 분류해줘.

[직접 경쟁사]
- 같은 문제를 같은 방식으로 해결하는 서비스
- 국내외 포함, 최대 5개

[간접 경쟁사]
- 같은 문제를 다른 방식으로 해결하거나
  사용자의 시간/돈을 경쟁하는 서비스
- 최대 3개

각 경쟁사마다:
- 서비스명
- 한 줄 설명
- 직접/간접 분류 이유

목록 작성 후, 내가 놓쳤을 것 같은 경쟁사가 있으면 추가로 제안해줘.
```

## [P-02] 경쟁사 심층 분석

> **목적:** 각 경쟁사의 강점과 약점을 7가지 기준으로 파악한다.

```
아래 경쟁사들을 7가지 기준으로 분석해줘.

분석 대상: [P-01에서 선택한 경쟁사 목록]

7가지 기준:
1. 핵심 기능 (무엇을 할 수 있는가)
2. 타겟 사용자 (누구를 위한 서비스인가)
3. 가격 구조 (무료/유료, 과금 방식)
4. 강점 (사용자가 선택하는 이유)
5. 약점 (리뷰나 커뮤니티에서 반복되는 불만)
6. 플랫폼 (앱/웹/오프라인)
7. 국내외 여부 (한국 사용자 접근 가능성)

결과를 표 형식으로 정리해줘.
분석이 끝나면:
- 경쟁사들의 공통 강점을 요약해줘 (내가 반드시 갖춰야 할 기본 조건)
- 경쟁사들의 공통 약점을 요약해줘 (내가 공략할 수 있는 틈새)
```

## [P-03] 시장 지도 & 포지셔닝 도출

> **목적:** 경쟁사 분석을 바탕으로 내 서비스의 포지셔닝 방향 3가지를 제안한다.

```
경쟁사 분석 결과를 바탕으로 포지셔닝 방향을 제안해줘.

Step 1. 시장 지도
- 이 시장에 적합한 2개의 축을 제안해줘
- 각 경쟁사를 그 축 위에 배치했을 때 빈 자리가 어디인지 설명해줘
- (텍스트로 표현 가능하면 표 또는 2x2 매트릭스 형태로)

Step 2. 포지셔닝 방향 3가지 제안
각 방향마다:
- 방향 이름 (한 줄 키워드)
- 포지셔닝 문장:
  "우리 서비스는 [타겟]을 위한 [카테고리]로,
  [경쟁사 또는 기존 방법]과 달리 [핵심 차별점]을 제공한다."
- 이 방향의 기회 요인
- 이 방향의 위험 요인

Step 3. 추천
3가지 중 1가지를 추천하고, 명확한 이유를 설명해줘.
내가 선택을 결정할 때까지 다음 단계로 넘어가지 말아줘.
```

## [P-04] 포지셔닝 확정 및 분석표 정리

> **목적:** 확정된 포지셔닝을 기반으로 경쟁사 분석표를 완성한다.

```
포지셔닝 방향 [선택한 방향 번호/이름]으로 확정할게.

이 방향을 기준으로:

1. 확정된 포지셔닝 문장을 최종 정리해줘
2. 이 포지셔닝에서 우리 서비스가 반드시 잘해야 하는 것 3가지를 뽑아줘
3. 이 포지셔닝에서 굳이 경쟁하지 않아도 되는 것을 정리해줘
4. templates/02-competitive-analysis.md 형식에 맞게
   경쟁사 분석표 전체를 정리해줘
```
